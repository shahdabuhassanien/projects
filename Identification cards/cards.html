HTML

Hyper Text Markup Language
It is a basic standard web language
Used to describe the structure of webpage.
"Backbone Of Web"

CSS

Cascading Stylesheet
It is considered as designing object for webpage.
In which it makes the webpage
Looks attractive and responsive.
"Skins for Webpage"

JAVASCRIPT

Javascript is considered as
Scripting language in which it is
used to make interaction of an webpage by making reactions on it.
"Interaction of Webpage"
